
# AI Diet Planner — Basic Version (A)

This is a simple, ready-to-run Streamlit application that:
- Calculates BMI, BMR and TDEE
- Uses a small Decision Tree model for diet recommendation
- Provides a friendly UI built with Streamlit

## Requirements
- Python 3.9+
- Install dependencies:
  ```bash
  pip install -r requirements.txt
  ```

## Run
```bash
streamlit run app.py
```
